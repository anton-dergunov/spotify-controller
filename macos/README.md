# Spotify Controller (macOS app)

SwiftUI player UI — standalone window for now; menu bar popover later.

## Requirements

- macOS 13+
- [Xcode](https://developer.apple.com/xcode/) or Xcode Command Line Tools (`xcode-select --install`)
- Album art at repo root: `../cover.jpg` (copied into the bundle on build)

## Build and run

From the **repository root**:

```bash
make run
```

Or from this directory:

```bash
make run
```

Other targets:

| Command | Action |
|---------|--------|
| `make build` | Release build only |
| `make debug` | Debug build and run |
| `make clean` | Remove build artifacts |

The app opens a window with the 300×300 player. **Hover** the square to show blurred cover, blue tint, and controls.
